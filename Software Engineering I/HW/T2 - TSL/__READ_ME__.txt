1. Use the directions at the bottom of category_partition_testing.docx
2. The input and output files are in the Spec folder. You can create your own folder, or simply put input/output files in the current folder.
3. To run TSL, in Windows open a command prompt and navigate to this folder.
3. Example call that should work:

// Shows results on screen

tsl -s Specs/getAid.txt

// Saves results to file

tsl Specs/getAid.txt -o Specs/getAid.txt.tsl
